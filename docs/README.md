# Backend Developer Portfolio

백엔드 개발자 포트폴리오 정적 웹사이트입니다.

---

## 🚀 완성된 기능

- **Hero 섹션** — 이름, 핵심 메시지, 기술 태그, CTA 버튼, 코드 블록 비주얼
- **소개 섹션** — 개발자 소개 문구, 주요 지표(수상 횟수, API 절감율, 자격증 수, TOEIC 점수)
- **기술 스택 섹션** — 핵심 6개 스택(Java, Spring Boot, Docker, MySQL, Redis, Python/FastAPI) + 기타 경험
- **프로젝트 섹션** — K-ing / OJO / PWAndora 3개 프로젝트 (역할, 기여, 문제 해결 상세 기술)
- **교육 및 자격증 섹션** — 타임라인 형태 교육 이력, 자격증 카드 목록
- **연락하기 섹션** — Email, GitHub, TEL 링크 카드
- **반응형 디자인** — 모바일/태블릿/데스크탑 대응
- **인터랙션** — 스크롤 네비게이션, Reveal 애니메이션, 스킬 바 애니메이션, 코드 커서 효과
